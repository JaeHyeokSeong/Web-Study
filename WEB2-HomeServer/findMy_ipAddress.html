IP 주소 알아내기 - private ip address

현재 위의 그림은 저의 컴퓨터의 네트워크 상태를 알려주고 있는 그림입니다. 여기에서 IP address 가 172.24.10.116 을 가지고 있는데 이것은 현재 나의 컴퓨터의 ip address 를 나타내고 있는 것 입니다. 그리고 Router 이라고 172.24.0.1 을 가지고 있는 항목이 보이는데 이것은 현재 집에 설치 되어진 router, 즉 공유기의 gateway address 를 가리키고 있는 것입니다.
