WEB2-Home Server

Port

컴퓨터 안에는 다양한 서버들이 존재 한다. 그러면 이때 컴퓨터에 들어온 신호는 어떠한 서버에 요청을 해야하는 것인지 아는 것일까? 어떠한 서버에 보내는지 알기위해서는 port 번호를 알아야 올바른 서버에 요청을 할 수 있는 것이다. 컴퓨터에는 총 65535 개의 port 번호들이 존재한다. 이때 0번 부터 1023번까지는 Well-known port 으로써 사전에 이미 예약이 되어진 포트번호들 이다. 예를들어서 port 22번은 SSH을 위해서 정해진 것이고, port 80번은 http, 즉 web server 를 위해서 예약되어진 port 번호 이다. 따라서 만약에 다른 서버를 설치하고 사용하고 싶다면 이미 예약되어진 번호들이 아닌 나머지 비여져 있는 port 번호들을 사용하면 된다.
