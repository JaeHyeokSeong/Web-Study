Dynamic VS Static IP Address

Dynamic IP Address

세상에는 한정되어진 ip address 를 가지고 있다. 따라서 이러한 문제점들을 해결하기 위해서 dynamic ip address 이라는 개념이 생기게 되었다. 만약에 A 이라는 집이 ISP (Internet Service Provider) 로 부터 즉 통신회사로 부터 ip 주소 59.6.66.238 을 받았다고 하자. 근데 A 이라는 집이 오래 기간동안 인터넷을 사용하지 않으면 IPS 는 이 ip address 를 다른 집 즉 새롭게 계약을 맺은 B 이라는 집에게 줄 것이다. 그리고 A 이라는 집이 다시 인터넷을 사용하기 시작하면 새로운 ip address 예를 들어서 69.8.61.138 을 주게 되는 것이다. 이를 통해서 ISP 는 한정되어진 ip address 를 여러사람들에게 이용할 수 있도록 줄수있다. 하지만 dynamic address 에도 문제점이 존재한다. 이는 만약에 A 이라는 집이 59.6.66.238 를 서버로서 사용하고 있다면 dynamic address 로 인해서 이 주소는 B 이라는 집에 부여가 된다. 따라서 신호가 A 이라는 집으로 안가게 되고 B 이라는 집의 주소로 가게 되는 것이다.

Static IP Address

Static IP Address 이란 Dynamic IP Address 이랑 달리 한번 부여되어진 ip address 가 변하지 않고 사용자가 받은 ip address 가 계속 유지 되어지는 것이다. 하지만 이는 dynamic ip address 보다는 통신 요금이 좀더 비싸다는 단점이 있다.
